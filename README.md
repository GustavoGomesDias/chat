## 2. 💻 Tecnologias usadas
* Node;
* SocketIO;
* Express;
* EJS

## 3. 🎉 Para rodar o projeto
1. Instale todas es tec's usadas junto do Node JS:

    ```
    npm install
    ```
2. Rode o aplicativo

    ```
    node index
    ```
3. Entre em https://localhost:3000 e teste o aplicativo
